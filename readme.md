Heart Failure Prediction App

This is a machine learning web application built with Streamlit to predict the risk of death due to heart failure based on clinical records.

The core prediction model is a Support Vector Classifier (SVC) trained on the Heart Failure Clinical Records Dataset.

🚀 How to Run Locally

Prerequisites

Python 3.8+

The following Python packages (see requirements.txt).

Setup

Clone the Repository:

git clone [https://github.com/your-username/heart-failure-prediction.git](https://github.com/your-username/heart-failure-prediction.git)
cd heart-failure-prediction


Install Dependencies:

pip install -r requirements.txt


Ensure Model is Present:
Make sure the heart_failure_deployment_artifact.pkl file (which contains the model, scaler, and features) is in the root directory. This file is generated by running train_model.py.

Run the Streamlit App:

streamlit run app.py


🧠 Model Details

Model Type: Support Vector Classifier (SVC) with RBF Kernel.

Preprocessing: Data is standardized using StandardScaler and key features (age, ejection_fraction, serum_sodium) are binned and one-hot encoded.

Training Script: train_model.py

Deployment Artifact: heart_failure_deployment_artifact.pkl

⚙️ Data Inputs Required

The app requires the user to input the following clinical parameters for prediction:

Age (years)

Anaemia (Boolean: 0/1)

Creatinine Phosphokinase (CPK) (mcg/L)

Diabetes (Boolean: 0/1)

Ejection Fraction (%)

High Blood Pressure (Boolean: 0/1)

Platelets (kiloplatelets/mL)

Serum Creatinine (mg/dL)

Serum Sodium (mEq/L)

Sex (Binary: 0=Female, 1=Male)

Smoking (Boolean: 0/1)

Time (Follow-up period, days)
